# Sentimetrics-

## Project Overview

Sentimetrics is a web-based mobile phone recommendation system developed as a Third Year Mini Project under the University of Mumbai. The project focuses on analyzing user preferences and sentiment-based metrics to provide structured and relevant mobile phone recommendations.

The system is built on insights collected through real-world surveys and field research, including interactions with consumers and visits to retail environments such as Reliance Digital. These inputs were used to design a practical and user-focused recommendation approach.

## Objective

To develop a platform that evaluates user requirements and suggests suitable mobile phones using structured comparison and sentiment-driven analysis.

## Key Features

* User preference-based mobile phone recommendations
* Sentiment and metric-based evaluation
* Comparative analysis of multiple devices
* Simple and interactive web interface
